# 单位换算器 — 迭代 0.0.1

## 范围

首个完整交付版本。实现长度、重量、温度三类单位换算的纯前端 SPA。

## 需求来源

- `e000001` owner_input: "做一个在线的单位换算器网页，支持长度(米/英尺/英寸/厘米/公里/英里)、重量(千克/磅/盎司/克)、温度(摄氏度/华氏度/开尔文)三种类型的单位换算。输入任意一个值自动实时换算出其他单位的结果。界面简洁好看，移动端友好。"
- `e000003` owner_confirmation

## 标准交付基线

- 前端 SPA（React/Vue + Vite）
- 三类单位换算完整实现
- 实时双向换算
- 响应式设计，移动端友好
- 自测脚本
- docs 同步

## 业务模型

| 类别 | 单位 |
|------|------|
| 长度 | 米(m)、英尺(ft)、英寸(in)、厘米(cm)、公里(km)、英里(mi) |
| 重量 | 千克(kg)、磅(lb)、盎司(oz)、克(g) |
| 温度 | 摄氏度(°C)、华氏度(°F)、开尔文(K) |

## 产品流程

- 主流程：打开页面 → 选择类别标签 → 在任一输入框键入数值 → 其他单位自动刷新
- 异常：非数字输入提示或忽略；开尔文 < 0 提示
